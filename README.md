\# VisaFlow — AI-Based Visa Processing System



A full-stack online visa application platform with AI-powered document verification.



\## Tech Stack

TypeScript, React, Node.js, Express, PostgreSQL, Supabase, Drizzle ORM, Gemini AI, Tailwind CSS



\## Features

\- Role-based access for Applicant, Officer, and Admin

\- AI-powered document verification and fraud detection using Gemini AI

\- Real-time application status tracking

\- Secure dashboard for managing users and applications
